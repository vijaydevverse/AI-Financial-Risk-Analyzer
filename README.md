# 💰 AI Financial Risk Analyzer

A machine learning-powered Streamlit web application that analyzes financial and loan data to predict credit risk. This project simulates a real-world banking risk assessment system using AI.

---

## 🚀 Live Features

- 📊 Upload any CSV dataset
- 🤖 Automatic data preprocessing
- 🧠 Machine Learning model (Random Forest)
- 📈 Interactive dashboard with charts
- 🔮 Real-time loan risk prediction
- 📉 Risk classification (High / Low)

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Plotly
